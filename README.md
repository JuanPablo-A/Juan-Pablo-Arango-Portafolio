## Getting Started
1. First clone the repoitory

    ```bash
    git clone https://github.com/JuanPablo-A/Juan-Pablo-Arango-Portafolio

    cd Juan-Pablo-Arango-Portfolio
    ```
2. Second insatall dependencies

    ```bash
    npm install
    ```

3. Run in local - Development mode 

    ```bash
    npm run dev 
    ```
